# 01 — Mockup: three-column PDF layout, paperclip menu, search dialog (HITL)

## What to build

A static design mockup (`design/mockup-pdf-layout.html`) showing the three newly
composed surfaces before any of them is implemented: (1) the three-column view with the
chat tree (connector lines) in the left sidebar, the PDF viewer center, and the active
branch's chat right; (2) the paperclip menu with "Upload file" and "Research paper"
options; (3) the paper-search dialog as a modal with the three availability states.
Follows the design language of the Syflo mockups copied into `design/`.

## Acceptance criteria

- [ ] All three surfaces visible in one self-contained HTML file
- [ ] Brand palette, Inter, soft cards, Lucide-style inline SVGs (no emojis), English copy
- [ ] **User has approved the mockup** (this is the HITL gate for issues 03–07)

## Blocked by

None - can start immediately
